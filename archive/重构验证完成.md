## 使用场景

### 场景 1: 在 Excel 里快速判断商品质量

打开 CSV，查看：
- `identical_ksp_terms` → 有多少热门搜索词
- `matched_ksp_terms` → 有多少有效 KSP
- `top_matched_searches` → 匹配的搜索词是否相关

### 场景 2: 生成 HTML 报告（简化版）

**原来**: 需要加载搜索词库（500MB）+ 实时查询每个 KSP
**现在**: 直接从 CSV 读取，无需加载搜索词库

```python
# 简化版代码
hit_7b = set(row["identical_ksp_terms_7b"].split("; "))
matched_7b = set(row["matched_ksp_terms_7b"].split("; "))
top_searches = row["top_matched_searches_7b"].split("; ")
```

**性能提升**:
- 省去加载搜索词库（0.02s + 500MB）
- 省去实时查询（每个 KSP 1 次查询）
- 总提升: ~1-2s + 500MB 内存

### 场景 3: 对比不同模型

直接对比两个模型的 CSV：
- `identical_ksp_terms` → 哪个模型生成了更多热门搜索词
- `top_matched_searches` → 哪个模型匹配到更相关的搜索词
- `matched_ksp_details` → 哪个模型的 KSP 贡献度更高
