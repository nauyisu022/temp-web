# 术语说明

## KSP 匹配的两种类型

### 1. ✓ 一致（Identical）

**定义**: KSP **本身就是**搜索词（完全相等或词干相等）

**判断方式**: `library.exact_lookup(ksp)`

**示例**:
```
KSP: "caderno universitario"
搜索词库中有: "caderno universitario"
→ ✓ 一致（identical）
```

**对应指标**: `identical_rate`（一致率）

**业务价值**: ⭐⭐⭐⭐⭐ 最高价值
- 说明 KSP 直接命中了用户的搜索意图
- 这些 KSP 本身就是热门搜索词

**CSV 列**: `identical_ksp_terms`

---

### 2. ≈ 命中（Matched）

**定义**: KSP **在搜索词中有匹配**（子串关系）

**判断方式**: `library.search(ksp)` 返回结果 > 0

**示例**:
```
KSP: "80 folhas"
搜索词库中有: "caderno 80 folhas", "caderno universitario 80 folhas"
→ ≈ 命中（matched，作为子串）
```

**对应指标**: `hit_rate_phrase`（短语命中率）

**业务价值**: ⭐⭐⭐ 中等价值
- 说明 KSP 覆盖了相关的搜索需求
- 包括用户加修饰词的搜索

**CSV 列**: `matched_ksp_terms`

---

## 关系

```
一致 ⊆ 命中

即：所有一致的 KSP 都是命中的，但不是所有命中的都是一致的
```

**示例**:
```
商品的 5 个 KSP:
  1. "caderno universitario" → ✓ 一致（本身就是搜索词）
  2. "80 folhas"            → ≈ 命中（在 "caderno 80 folhas" 中）
  3. "capa dura"            → ≈ 命中（在 "fichario capa dura" 中）
  4. "espiral"              → ≈ 命中（在 "caderno espiral" 中）
  5. "marca X"              → ✗ 无匹配

统计:
  - identical_ksp_terms: "caderno universitario" (1 个)
  - matched_ksp_terms: "caderno universitario; 80 folhas; capa dura; espiral" (4 个)
  - identical_rate: 1/5 = 20%
  - hit_rate_phrase: 4/5 = 80%
```

## 在 HTML 报告中的显示

### 视觉标记

| 标记 | 样式 | 含义 |
|------|------|------|
| ✓ 一致 | 橙色边框 | KSP 本身就是搜索词 |
| ≈ 命中 | 蓝色边框 | KSP 在搜索词中有匹配 |
| 绿色背景 | 绿色背景 | 两个模型都生成了这个 KSP |

### 组合效果

- **绿色背景 + 橙色边框 + ✓ 一致** = 两个模型都生成了，且是热门搜索词
- **绿色背景 + 蓝色边框 + ≈ 命中** = 两个模型都生成了，且有搜索匹配
- **橙色边框 + ✓ 一致** = 是热门搜索词（只有一个模型生成）
- **蓝色边框 + ≈ 命中** = 有搜索匹配（只有一个模型生成）
- **无标记** = 没有搜索匹配

## 实际案例

### 案例 1: 高质量商品

```
商品: Caderno Universitario 80 Folhas

一致的 KSP (2 个):
  ✓ caderno universitario
  ✓ 80 folhas

命中的 KSP (6 个):
  ✓ caderno universitario (一致)
  ✓ 80 folhas (一致)
  ≈ capa dura (命中)
  ≈ espiral (命中)
  ≈ papel sulfite (命中)
  ≈ 96 folhas (命中)

统计:
  - 一致率: 2/8 = 25%
  - 命中率: 6/8 = 75%
```

**分析**: 
- ✅ 有 2 个热门搜索词（高价值）
- ✅ 有 6 个覆盖搜索需求（覆盖面广）
- **结论**: 高质量商品

---

### 案例 2: 低质量商品

```
商品: Placa Petri De Vidro 100x20mm

一致的 KSP (1 个):
  ✓ placa de petri

命中的 KSP (3 个):
  ✓ placa de petri (一致)
  ≈ laboratório (命中)
  ≈ microbiologia (命中)

统计:
  - 一致率: 1/16 = 6.3%
  - 命中率: 3/16 = 18.8%
```

**分析**:
- ⚠️ 只有 1 个热门搜索词（价值低）
- ⚠️ 只有 3 个覆盖搜索需求（覆盖面窄）
- **结论**: 低质量商品，需要优化

---

## 总结

| 术语 | 含义 | 价值 | 判断方式 |
|------|------|------|---------|
| **一致** | KSP = 搜索词 | ⭐⭐⭐⭐⭐ | `exact_lookup()` |
| **命中** | KSP in 搜索词 | ⭐⭐⭐ | `search()` |

**记忆方法**:
- **一致** = 更严格（完全相等）→ 更高价值
- **命中** = 更宽松（子串关系）→ 覆盖面广
